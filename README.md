## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install
```

Then start both the client and server 

```bash
npm start
```

## Usage

```javascript
import foobar

# returns 'words'
foobar.pluralize('word')
```
